## 🚀 Quick Start

1. Double-click `RankBlitz.bat`
2. Open Pet Simulator 99 in Roblox
3. Set your keybinds in the **Keybinds** tab to match your in-game equipment bar
4. Configure gamepasses and timing in the **Settings** tab
5. Press **F9** or click **Start**

## 🔧 Troubleshooting

**White screen / UI doesn't load**  
→ Install the Edge WebView2 Runtime (link above).

**Clicks land in wrong spots**  
→ Roblox must be in windowed mode. The macro uses client-relative coordinates. Tune pixel coords in `RankBlitz.ahk` for your resolution.

**Delay too fast/slow**  
→ Increase `delayModifier` in the **Settings** tab (e.g. 1.5 for laggy connections).
